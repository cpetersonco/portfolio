# Portfolio

A personal website for highlighting skills and professional development through
side projects and blog posts. Built with [Astro](https://astro.build/),
[Tailwind CSS](https://tailwindcss.com/), and MDX.

## Installation

```bash
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

Other commands:

| Command           | Action                          |
| ----------------- | ------------------------------- |
| `npm run build`   | Build for production to `dist/` |
| `npm run preview` | Preview the production build    |
| `npm run check`   | Run Astro type checking         |
| `npm run lint`    | Lint with ESLint                |
